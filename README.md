# Clog

Zero-config Kotlin multiplatform logging utility

## Supported Platforms/Features

| Platform | Logging Target   | ANSI Colors | Tag Inference | SLF4j Integration |
| -------- | ---------------- | ----------- | ------------- | ----------------- |
| JVM      | Console/stdout   | ✅          | ✅             | ✅                |
| Android  | android.util.Log | ❌          | ✅             | ❌                |
| JS       | console.log      | ❌          | ❌             | ❌                |
| iOS      | NSLog            | ❌          | ❌             | ❌                |
